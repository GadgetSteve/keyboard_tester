# Installation

## Python & wxPython Installation

If python is not already installed then it can be downloaded from [the Python web site](https://www.python.org/downloads/). Once it is installed use  `pip install wxpython` to get the required libray on an internet connected machine.

## Python & wxPython Installation On Windows 7

The last pre-compiled python to support Windows 7 was 3.8.10 - the installer can be downloaded for [64-bit](https://www.python.org/ftp/python/3.8.10/python-3.8.10-amd64.exe) or [32-bit](https://www.python.org/ftp/python/3.8.10/python-3.8.10.exe) Windows.

Once python is installed the wxPython library needs to be installed with `pip install wxpython==4.0.7` on an internet connected machine.
